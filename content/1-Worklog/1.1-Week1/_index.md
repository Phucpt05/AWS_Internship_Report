### Week 1 Achievements:

* Connected and got acquainted with members of First Cloud Journey, and understood the rules and regulations of the internship unit.

* Understood and categorized basic AWS services:
  * **Compute** (EC2, Lambda)
  * **Storage** (EBS, S3)
  * **Networking** (VPC, Elastic IP)
  * **Database** (RDS)

* Successfully created and configured an **AWS Free Tier account**, and set up **AWS Budgets** to manage service costs.

* Learned about **AWS Identity and Access Management (IAM)**:
  * Created and managed users
  * Assigned basic permissions

* Became familiar with the **AWS Management Console** and learned how to search, access, and use AWS services via the web interface.

* Installed and configured **AWS CLI** on the computer:
  * Configured Access Key, Secret Key, and Default Region
  * Performed basic operations such as:
    * Checking account and configuration information
    * Retrieving the list of regions
    * Managing key pairs
    * Querying running services

* Practiced deploying AWS resources:
  * Launched an EC2 instance
  * Connected via SSH
  * Attached and managed an EBS volume

* Deployed sample applications:
  * Created a simple Lambda function
  * Set up a database using Amazon RDS

* Acquired the ability to manage AWS resources in parallel through both the **Console** and the **CLI**.
