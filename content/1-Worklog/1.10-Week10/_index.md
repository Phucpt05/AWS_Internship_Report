### Week 10 Objectives:

* Learn how to monitor and analyze AWS resources using **CloudWatch** and **CloudTrail**.  
* Understand AWS cost management tools: **Cost Explorer** and **Budgets**.  
* Explore the concept of **Infrastructure as Code (IaC)** with **CloudFormation**.  
* Practice automating infrastructure deployment and cleanup workflows.  

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | ---- | ----------- | ---------------- | ------------------ |
| 1 | - Study **Amazon CloudWatch Metrics** and create custom dashboards. <br> - **Practice:** Set alarms for EC2 instance CPU utilization. | 10/11/2025 | 10/11/2025 | [AWS CloudWatch Docs](https://docs.aws.amazon.com/cloudwatch/) |
| 2 | - Learn **CloudTrail** for tracking AWS API activity. <br> - **Practice:** Query and analyze logs to identify recent user actions. | 11/11/2025 | 11/11/2025 | [AWS CloudTrail Docs](https://docs.aws.amazon.com/cloudtrail/) |
| 3 | - Explore **Cost Explorer** for billing and usage insights. <br> - **Practice:** Analyze cost by service and region, identify top resource spenders. | 12/11/2025 | 12/11/2025 | [AWS Cost Explorer Guide](https://docs.aws.amazon.com/cost-management/latest/userguide/ce-what-is.html) |
| 4 | - Learn about **AWS Budgets**. <br> - **Practice:** Set cost budget and create email alert thresholds. | 13/11/2025 | 13/11/2025 | [AWS Budgets Docs](https://docs.aws.amazon.com/cost-management/latest/userguide/budgets-managing-costs.html) |
| 5 | - Introduction to **CloudFormation**. <br> - **Practice:** Deploy and delete stacks using CloudFormation templates. | 14/11/2025 | 14/11/2025 | [AWS CloudFormation Docs](https://docs.aws.amazon.com/cloudformation/) |
| 6 | - **Advanced Monitoring:** CloudWatch Logs and Insights <br> - Create log groups and filter log data <br> - Set up automated responses to alarms | 15/11/2025 | 15/11/2025 | [AWS CloudWatch Logs](https://docs.aws.amazon.com/cloudwatch/) |
| 7 | - **Infrastructure as Code Advanced:** Nested stacks and drift detection <br> - Implement change sets for safe updates <br> - Practice resource cleanup automation | 16/11/2025 | 16/11/2025 | [AWS CloudFormation Advanced](https://cloudjourney.awsstudygroup.com/) |

### Week 10 Achievements:

* Successfully monitored EC2 instances and Lambda metrics with **CloudWatch**, and created custom dashboards to visualize resource usage.  
* Configured **CloudWatch alarms** to automatically trigger notifications when system thresholds are exceeded.  
* Used **CloudTrail** to review activity logs, ensuring visibility into user actions and API calls.  
* Gained experience analyzing AWS spending with **Cost Explorer** and configuring **Budgets** for proactive cost management.  
* Created and deployed infrastructure resources using **CloudFormation templates**, applying the **Infrastructure as Code (IaC)** principle.  
* Automated both creation and cleanup of resources to optimize workflow efficiency.  
* Strengthened understanding of AWS monitoring, auditing, and automation as core components of cloud governance.
